# Service Categories

Organize your services into categories for easier management.

[image_service_categories_list]

## Overview

Service categories help organize your service offerings. Categories can be nested (parent-child) for hierarchical organization.

## Viewing Categories

Navigate to **Admin > Services > Categories** to see all categories.

[image_service_categories_index]

## Creating a Category

1. Navigate to **Admin > Services > Categories**
2. Fill in the form on the left side
3. Click **Save**

[image_create_service_category]

### Category Fields

| Field | Description | Required |
|-------|-------------|----------|
| **Name** | Category name | Yes |
| **Parent** | Parent category (for nesting) | No |
| **Description** | Category description | No |
| **Status** | Active or Inactive | Yes |

## Nested Categories

Create a hierarchy by setting a parent category:

```
Screen Repairs
├── Phone Screen Repairs
└── Tablet Screen Repairs

Battery Replacements
├── Phone Batteries
└── Laptop Batteries
```

## Editing a Category

1. Find the category in the list
2. Click **Edit** button
3. Modify the information
4. Click **Save**

## Deleting a Category

1. Find the category in the list
2. Click **Delete** button
3. Confirm deletion

> **Warning:** Deleting a parent category will also delete all child categories. Services in deleted categories will become uncategorized.

## Using Categories

### Filtering Services

On the Services list page, use the category filter to show only services in a specific category.

### Assigning Services to Categories

When creating or editing a service, select the category from the dropdown.
