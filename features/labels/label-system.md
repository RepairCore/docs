# Labels

Print labels for devices and repair orders.

[image_labels_page]

## Overview

Labels help identify devices in your shop. Print labels with barcodes, order codes, and customer information.

## Printing Labels

### From Repair Order

1. Open a repair order
2. Go to **Labels** tab
3. Click **Print Label**
4. Label opens in new tab
5. Print

[image_print_label_button]

### Label Contents

Labels typically include:
- Order code (with barcode)
- Customer name
- Device information
- Problem summary
- Date received
- Priority indicator

[image_label_example]

## Label Sizes

Common label sizes:
- **Small** - 2" x 1" (for small devices)
- **Medium** - 3" x 2" (standard)
- **Large** - 4" x 3" (for detailed info)

## Printing Tips

### Printer Setup

1. Use a label printer or standard printer
2. Load appropriate label paper
3. Adjust printer settings to match label size

### Browser Settings

When printing:
1. Set margins to "None" or "Minimum"
2. Disable headers and footers
3. Set scale to 100%

## Using Labels

### Attach to Device

Attach the label to the device or its case/bag for easy identification.

### Scanning

If you have a barcode scanner:
1. Scan the label barcode
2. System opens the repair order
3. Quick access to order details

## Multiple Labels

To print multiple labels:
1. Open the repair order
2. Go to **Labels** tab
3. Select quantity
4. Click **Print**
