# User Settings

Configure user registration and authentication settings.

[image_user_settings_page]

## Overview

User settings control how users register, log in, and manage their accounts in the system.

## Accessing User Settings

Navigate to **Settings > User** in the admin panel.

## Registration Settings

| Setting | Description |
|---------|-------------|
| **Allow Registration** | Enable/disable public user registration |
| **Customer Role** | Role assigned to customers created from repair orders |

## Saving Changes

1. Modify user settings
2. Click **Save Settings**
3. Changes apply to new sessions
