# Navigation

Learn how to navigate the RepairCore admin panel.

[image_admin_panel_layout]

## Layout Overview

The admin panel consists of:
- **Header** - Top bar with user menu and notifications
- **Sidebar** - Left navigation menu
- **Content Area** - Main working area

## Header

[image_admin_header]

### User Menu

Click your name/avatar in the top right to access:
- **Profile** - Edit your profile
- **Settings** - Quick access to settings
- **Logout** - Sign out

### Notifications

Click the bell icon to view notifications:
- New order alerts
- Assignment notifications
- System messages

## Sidebar Menu

The sidebar provides access to all features:

[image_sidebar_menu]

### Main Menu Items

| Menu | Description |
|------|-------------|
| **Dashboard** | Overview and statistics |
| **Repair Orders** | Manage repair orders |
| **Customers** | Customer management |
| **Devices** | Device catalog |
| **Inventory** | Parts and stock |
| **Services** | Service catalog |
| **Users** | User management |
| **Labels** | Label management |
| **Settings** | System configuration |

### Collapsible Sections

Some menu items expand to show sub-items:
- Click the arrow to expand/collapse
- Sub-items appear indented below

### Active Page

The current page is highlighted in the sidebar.

## Breadcrumbs

Below the header, breadcrumbs show your current location:

```
Dashboard > Repair Orders > Edit Order
```

Click any breadcrumb to navigate back.

[image_breadcrumbs]

## Content Area

The main content area displays:
- **Page Title** - Current page name
- **Action Buttons** - Create, Save, Delete, etc.
- **Content** - Forms, tables, details

## Common Actions

### Creating Items

1. Navigate to the section (e.g., Customers)
2. Click **Create** or **Add New** button
3. Fill in the form
4. Click **Save**

### Editing Items

1. Find the item in the list
2. Click **Edit** button or the item name
3. Modify the form
4. Click **Save**

### Deleting Items

1. Find the item in the list
2. Click **Delete** button
3. Confirm deletion

### Bulk Actions

1. Select items using checkboxes
2. Choose action from dropdown
3. Click **Apply**

[image_bulk_actions]

## Search

Most list pages have a search box:
1. Type your search term
2. Press Enter or click Search
3. Results are filtered

## Pagination

Large lists are paginated:
- Use page numbers to navigate
- Change items per page if available
